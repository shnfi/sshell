## purpose ?

## how to run ?
just compile the `main.c` file . (do not forget the `lncurses` option!)
```bash
gcc main.c -lncurses -o sshell
```
and then run the `sshell` file .
```bash
./sshell
```

<br>OR..<br><br>

just use the `compile.sh` file .
```bash
chmod +x compile.sh && ./compile.sh
```
and then run the `sshell` file .
```bash
./sshell
```
